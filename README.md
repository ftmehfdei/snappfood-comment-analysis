# snappfood-comment-analysis
Persian SnappFood Sentiment Analysis: NLP project classifying 70K Persian reviews into HAPPY/SAD. Custom preprocessing + TF-IDF + tuned Logistic Regression achieving 82%+ accuracy. Top indicators: "ممنون/خوشمزه" (positive), "سرد/افتضاح" (negative). Built with Python/Scikit-learn. Complete ML workflow from cleaning to deployment.
